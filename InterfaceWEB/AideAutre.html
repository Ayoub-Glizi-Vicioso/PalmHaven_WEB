Comment laisser une revue?

Afin de laisser une revue, il suffit d’accéder à l’onglet review se trouvant dans le pied-de-page de la page d’accueil. Le site vous amènera dans une nouvelle page contenant toutes les revues sur le site d’autres clients, les vôtres et une section contenant un forum vous permettant de laisser votre avis.

Comment supprimer une revue?

Vous ne pouvez que supprimer votre propre avis. Pour ce faire, allez dans l’onglet review dans le pied-de-page de la page d’accueil, proche de vos commentaires, une option de suppression sera disponible.

Note : un message de confirmation de suppression s’affichera.

Comment supprimer votre compte?

Pour ce faire, allez sur votre profil et une option de suppression sera disponible.

Note : un message de confirmation de suppression s’affichera.

Comment savoir que mes informations sont protégées?

Nous tenons à cœur la sécurité personnelle de nos utilisateurs, c’est pourquoi nous faisons tout pour crypter vos données les plus précieuses.
